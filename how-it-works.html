What it does

Presence in one tap.

Residue is a laser-engraved, stainless-steel NFC card. Tap it on a phone and it launches your page with your name, phone, email, socials, site, map pin, reel, and any link you curate.

You never ask for a number or spell out handles—the card handles the exchange so you can stay in the moment.

Access

How to get a card.

Supply stays tight. Each purchase uses one access code. A code is single-use per checkout, but you can place more than one card in that checkout. Every card issued comes with its own fresh code you can pass to one other person.

Hidden codes: Some codes live on stickers in the wild. If you’re here from a sticker, you’ve found one—use it once.

Circulation windows: At times, we open a 24–48 hour window for anyone. Countdown and status live on the Circulation page.

Access requests: You can request a code. We review intent, demand, and fit. Approved requests receive a code that unlocks one checkout.

This is intentional scarcity—more Rolls than rideshare. The value is in who carries it and how it’s used.
